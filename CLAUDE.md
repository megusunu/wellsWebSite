# Wells合同会社ホームページ プロジェクト

## プロジェクト概要

Wells合同会社の公式ホームページ。システム開発とハンドメイド小物販売の二つの事業を紹介するモダンなWebサイト。

### 技術スタック
- **フレームワーク**: Astro
- **言語**: TypeScript
- **CSS**: TailwindCSS
- **開発サーバー**: http://localhost:9000/

## ビジネス要件

### 会社情報
- **会社名**: Wells合同会社
- **事業内容**: 
  1. システム開発（React、Vue.js、React Native、Swift、Android Studio）
  2. ハンドメイド小物販売（編み物、ペット用品、季節小物）

### 外部販売サイト
- **Square**: https://megusunulab.square.site/
- **minne**: https://minne.com/@megusunu0120

## デザインコンセプト

### カラーパレット
```css
colors: {
  'natural-green': '#7CB342',
  'natural-light-green': '#AED581', 
  'natural-beige': '#F5E6D3',
  'natural-brown': '#8D6E63',
  'warm-yellow': '#FFD54F',
  'vibrant-orange': '#FF7043',    // ポイントカラー
  'light-orange': '#FFAB91',
}
```

### デザイン原則
- **ナチュラル**: 緑・ベージュを基調とした温かみのある配色
- **ポジティブ**: オレンジをアクセントカラーとして使用
- **モダン**: ゴシック体でスタイリッシュな印象
- **シンプル**: 不要な装飾を避けたクリーンなデザイン

## サイト構造

### ページ構成
1. **ヒーローセクション** (`Hero.astro`)
   - 会社ロゴマーク
   - 会社名（ゴシック体、白文字、ドロップシャドウ）
   - キャッチフレーズ
   - CTAボタン

2. **システム開発** (`Services.astro`)
   - 技術スタック紹介
   - サービス内容
   - 実績紹介

3. **ハンドメイド作品** (`Handmade.astro`)
   - 商品スライドショー
   - 価格情報
   - 外部ショップリンク

4. **お問い合わせ** (`Contact.astro`)
   - コンタクトフォーム

5. **フッター** (`Footer.astro`)
   - 会社情報
   - 外部リンク

### コンポーネント構造
```
src/
├── components/
│   ├── Header.astro          # ナビゲーション
│   ├── Hero.astro           # ヒーローセクション
│   ├── Services.astro       # システム開発
│   ├── Handmade.astro       # ハンドメイド
│   ├── Contact.astro        # お問い合わせ
│   ├── Footer.astro         # フッター
│   └── ImageSlideshow.astro # スライドショー
├── layouts/
│   └── Layout.astro         # ベースレイアウト
└── pages/
    └── index.astro          # メインページ
```

## スライドショー機能

### 仕様
- **自動再生**: 4秒間隔
- **ナビゲーション**: 左右矢印ボタン
- **インジケーター**: ドット（オレンジでアクティブ表示）
- **ホバー制御**: マウスオーバーで自動再生停止
- **複数インスタンス**: 独立した動作

### 商品画像
```
public/images/products/
├── pet-wear/           # ペット用ニットウェア (5枚)
├── wrist-warmers/      # リストウォーマー (4枚)
└── tulips/            # 編みぐるみ・季節小物 (4枚)
```

## 開発ルール

### フォント使用
- **ヘッダー**: `font-sans font-black` (ゴシック体・極太)
- **本文**: `font-sans` (ゴシック体)
- **統一原則**: サイト全体でゴシック体を使用

### カラー使用原則
- **メインカラー**: natural-green
- **アクセント**: vibrant-orange
- **背景**: natural-beige系
- **テキスト**: natural-brown、グレー系

### レスポンシブ対応
- **ブレークポイント**: Tailwind標準（sm、md、lg、xl）
- **画像**: srcset対応で高解像度対応
- **レイアウト**: モバイルファースト

### パフォーマンス
- **画像最適化**: 複数解像度対応
- **ローカル管理**: 外部依存を最小化
- **遅延読み込み**: `loading="lazy"`
